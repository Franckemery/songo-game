📖 Règles du Songo

🎯 Objectif

Capturer plus de 40 graines avant l'adversaire, ou avoir le plus de graines quand la partie s'arrête.

🔄 Distribution

Tu récupères toutes les graines d'une case et tu les déposes une à une dans le sens circulaire :

S1 → S2 → S3 → S4 → S5 → S6 → S7 → N7 → N6 → N5 → N4 → N3 → N2 → N1 → S1…

⚠️ La case de départ est ignorée si on repasse dessus.

🎯 Capture

La dernière graine tombe dans une case adverse contenant après dépôt 2, 3 ou 4 graines → capture !

⛓️ Capture en chaîne

Après une capture, on examine les cases précédentes dans le camp adverse. Si elles contiennent 2, 3 ou 4 graines → capturées aussi. On s'arrête dès qu'une case ne remplit pas la condition.

🚫 Interdictions

❌ Jouer une case vide.
❌ Capturer si cela vide tout le camp adverse (la capture est annulée).
❌ Ne pas nourrir l'adversaire quand il n'a plus de graines.

🏁 Fin de partie

✅ Un joueur atteint 40 graines capturées → victoire immédiate.
✅ Moins de 10 graines sur le plateau → fin.
✅ Impossible de nourrir l'adversaire → fin.
